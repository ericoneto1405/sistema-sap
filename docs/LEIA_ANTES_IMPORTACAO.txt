  PASSO 1: VERIFICAR IDs DE CLIENTES E PRODUTOS NO SISTEMA
═══════════════════════════════════════════════════════════════════════════════

Antes de preencher o arquivo de importação, você PRECISA saber os IDs corretos
dos seus clientes e produtos que já estão cadastrados no sistema.

🔍 COMO ENCONTRAR OS IDs:

  1. Para CLIENTES:
     • Acesse: Menu → Clientes → Listar
     • A primeira coluna (#) mostra o ID de cada cliente
     • Anote os IDs que você vai usar

  2. Para PRODUTOS:
     • Acesse: Menu → Produtos → Listar
     • A primeira coluna (#) mostra o ID de cada produto
     • Anote os IDs que você vai usar

═══════════════════════════════════════════════════════════════════════════════
  PASSO 2: ENTENDER O ARQUIVO DE EXEMPLO
═══════════════════════════════════════════════════════════════════════════════

O arquivo EXEMPLO_IMPORTACAO_PEDIDOS.csv contém dados fictícios:

  cliente_id,produto_id,quantidade,preco_venda,data
  1,1,10,25.50,2024-01-15
  1,2,5,15.00,2024-01-15
  2,1,20,8.75,2024-01-16

📌 O QUE SIGNIFICA:

  • Linha 2-3: Cliente #1 fez um pedido em 15/01/2024
               - 10 unidades do Produto #1 a R$ 25,50
               - 5 unidades do Produto #2 a R$ 15,00
               (Essas 2 linhas viram 1 pedido com 2 itens)

  • Linha 4: Cliente #2 fez um pedido em 16/01/2024
             - 20 unidades do Produto #1 a R$ 8,75
             (Esta linha vira 1 pedido com 1 item)

═══════════════════════════════════════════════════════════════════════════════
  PASSO 3: PREPARAR SEU ARQUIVO
═══════════════════════════════════════════════════════════════════════════════

✏️  EDITE O ARQUIVO CSV:

  1. Abra EXEMPLO_IMPORTACAO_PEDIDOS.csv no Excel, LibreOffice ou Google Sheets

  2. SUBSTITUA os IDs de exemplo pelos IDs reais do seu sistema:
     
     ❌ ERRADO (IDs fictícios):
        1,1,10,25.50,2024-01-15
     
     ✅ CORRETO (IDs do seu sistema):
        5,12,10,25.50,2024-01-15
        (Se Cliente José tem ID=5 e Produto Caneta tem ID=12)

  3. Ajuste as QUANTIDADES e PREÇOS conforme necessário

  4. Ajuste as DATAS para refletir quando os pedidos foram feitos

  5. Salve o arquivo como CSV

═══════════════════════════════════════════════════════════════════════════════
  FORMATO DE CADA COLUNA
═══════════════════════════════════════════════════════════════════════════════

📊 COLUNAS OBRIGATÓRIAS:

  ┌─────────────┬──────────────┬─────────────────────────────────────────────┐
  │ COLUNA      │ TIPO         │ EXEMPLOS VÁLIDOS                            │
  ├─────────────┼──────────────┼─────────────────────────────────────────────┤
  │ cliente_id  │ Número       │ 1, 5, 12, 999                               │
  │             │ inteiro      │                                             │
  ├─────────────┼──────────────┼─────────────────────────────────────────────┤
  │ produto_id  │ Número       │ 1, 3, 8, 150                                │
  │             │ inteiro      │                                             │
  ├─────────────┼──────────────┼─────────────────────────────────────────────┤
  │ quantidade  │ Número       │ 1, 10, 50, 1000                             │
  │             │ inteiro      │                                             │
  ├─────────────┼──────────────┼─────────────────────────────────────────────┤
  │ preco_venda │ Número       │ 10.50, 25.00, 99.99                         │
  │             │ decimal      │ (use PONTO como separador decimal)          │
  ├─────────────┼──────────────┼─────────────────────────────────────────────┤
  │ data        │ Data         │ 2024-01-15                                  │
  │             │              │ 2024-12-25                                  │
  │             │              │ 15/01/2024 (também funciona)                │
  └─────────────┴──────────────┴─────────────────────────────────────────────┘

⚠️  ATENÇÃO AO FORMATO DOS PREÇOS:
    • Use PONTO (.) como separador decimal: 25.50
    • NÃO use vírgula: ❌ 25,50
    • NÃO use símbolo R$: ❌ R$ 25.50
    • NÃO use separador de milhares: ❌ 1.000.50

═══════════════════════════════════════════════════════════════════════════════
  PASSO 4: AGRUPAR ITENS EM UM PEDIDO
═══════════════════════════════════════════════════════════════════════════════

🔄 REGRA DE AGRUPAMENTO:

  Linhas com MESMO cliente_id + MESMA data = 1 PEDIDO com múltiplos itens

  EXEMPLO:
    
    Cliente 5 em 15/01/2024:
    5,10,20,15.00,2024-01-15  ┐
    5,12,10,25.50,2024-01-15  ├─→ VIRA 1 PEDIDO com 2 itens
    5,8,5,8.75,2024-01-15     ┘

    Cliente 5 em 16/01/2024:
    5,10,15,15.00,2024-01-16  ──→ VIRA 1 PEDIDO com 1 item

    Resultado: 2 pedidos criados para o Cliente 5

═══════════════════════════════════════════════════════════════════════════════
  PASSO 5: IMPORTAR NO SISTEMA
═══════════════════════════════════════════════════════════════════════════════

📤 PROCESSO DE IMPORTAÇÃO:

  1. Acesse: Menu → Pedidos → Importar Histórico
  
  2. Clique em "Clique para selecionar" OU arraste o arquivo
  
  3. Clique em "Importar Pedidos"
  
  4. Aguarde o processamento
  
  5. Verifique a mensagem de sucesso/erro
  
  6. Confira os pedidos em: Menu → Pedidos → Listar

═══════════════════════════════════════════════════════════════════════════════
  ❓ PERGUNTAS FREQUENTES
═══════════════════════════════════════════════════════════════════════════════

Q: Posso usar Excel em vez de CSV?
A: Sim! O sistema aceita .xlsx e .xls também.

Q: E se um cliente ou produto não existir?
A: A linha será ignorada e você verá um aviso. Os demais pedidos serão importados.

Q: Como sei se deu certo?
A: Você verá uma mensagem verde com "X pedido(s) importado(s) com sucesso!"

Q: Posso importar o mesmo pedido duas vezes?
A: Sim, mas isso criará duplicatas. Cuidado!

Q: O que acontece se eu errar um preço?
A: Você pode editar o pedido depois da importação.

Q: Preciso importar os pagamentos também?
A: Não. Os pagamentos devem ser adicionados manualmente depois.
